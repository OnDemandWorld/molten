# Release Notes - Molten v1.0.0

## What's New

**Initial Release** 🎉

Molten brings private, offline AI to your Mac, iPhone, and iPad.

### Features

✨ **Multi-Provider Support**
- Ollama integration
- Swama integration (MLX-optimized)
- Apple Foundation Models (macOS 26.0+)

🔒 **Privacy First**
- 100% offline processing
- No data collection
- No telemetry or tracking
- Open source & auditable

⚡ **Performance**
- Optimized for Apple Silicon
- Real-time streaming responses
- Performance analytics (tokens/sec)

💬 **Chat Experience**
- Beautiful native SwiftUI interface
- Conversation history
- Markdown rendering with code highlighting
- Voice input & text-to-speech

🎨 **Design**
- Dark/Light mode
- Keyboard shortcuts (⌘↩, ⌘⌥K)
- Floating panel mode
- Responsive on all screen sizes

### Technical

- Swift 6 concurrency
- SwiftData persistence
- 100% local processing
- No external dependencies at runtime

---

**Requirements**: macOS 14.0+, iOS 17.0+, iPadOS 17.0+  
**Note**: Requires Ollama or Swama installed for model access (free, open-source)

**Privacy**: Molten collects no data. Ever. See privacy policy for details.
