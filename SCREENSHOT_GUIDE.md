## How to Capture Screenshots

### Method 1: macOS Screenshot Tool

1. Press `Cmd + Shift + 4`
2. Press `Spacebar` (cursor becomes camera)
3. Click app window
4. Screenshot saved to Desktop

### Method 2: Xcode Simulator

**For iOS/iPadOS screenshots:**

1. Open Simulator with desired device
2. Run Molten in Simulator
3. Press `Cmd + S` to save screenshot
4. Screenshot saved to Desktop

**Advantages:**
- Exact device sizes
- Clean device frames
- No menu bar or dock

### Method 3: Real Device

**For iOS screenshots:**

1. Connect iPhone/iPad to Mac
2. Open Molten on device
3. In Xcode: Window → Devices and Simulators
4. Select device, click "View Device Logs"
5. Take screenshot from device

Or use device buttons:
- **iPhone with Face ID**: Side button + Volume up
- **iPhone with Home button**: Side button + Home button

## Editing Screenshots

### Recommended Tools

- **Preview** (built-in) - Crop, resize, annotate
- **Photoshop** - Professional editing
- **Canva** - Free online editor with App Store templates
- **Figma** - Free design tool

### Editing Tips

1. **Crop tightly** - Remove unnecessary desktop space
2. **Maintain aspect ratio** - Don't stretch
3. **Add device frames** (optional) - Use Apple's device frame templates
4. **Consistent styling** - All screenshots in same mode (dark or light)
5. **High contrast** - Ensure text is readable

### What NOT to Do

- ❌ Don't add text overlays (App Store adds captions)
- ❌ Don't use simulator bezels (use clean screenshots)
- ❌ Don't include menu bar or dock
- ❌ Don't blur or pixelate content
- ❌ Don't use screenshots from other apps

## App Preview Video (Optional)

### Requirements

- **Length**: 15-30 seconds (recommended)
- **Maximum**: 30 seconds
- **Format**: MP4 or MOV
- **Resolution**: 1920x1080 or higher
- **Audio**: Optional (no narration needed)

### Recommended Storyboard

**0:00-0:05** - App launch, show main interface  
**0:05-0:10** - Select model from dropdown  
**0:10-0:15** - Type a prompt and send  
**0:15-0:20** - Show streaming response with analytics  
**0:20-0:25** - Show stop button, press it  
**0:25-0:30** - Show conversation history sidebar  

### Recording Tools

- **QuickTime Player** (built-in) - Screen recording
- **ScreenFlow** - Professional screen recording
- **OBS Studio** - Free screen recording
- **Final Cut Pro** - Video editing

### Tips

- Keep it simple and focused
- Show key features quickly
- No need for audio narration
- Use smooth transitions
- Export at 1080p minimum

## Uploading to App Store Connect

### Steps

1. Go to App Store Connect
2. Select Molten
3. Click version 1.0.0
4. Scroll to "Screenshots" section
5. Select device type (macOS, iOS, iPadOS)
6. Drag and drop screenshots
7. Add captions (optional but recommended)
8. Reorder by dragging

### Caption Examples

- "Chat with Local AI Models"
- "Multi-Provider Support"
- "100% Offline & Private"
- "Beautiful Native Design"
- "Real-Time Performance Metrics"
- "Dark Mode Included"
