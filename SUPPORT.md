# Support

Molten is an open source project and collaborators are very welcome.

For support please create a Github issue or contact the maintainers.

## Common Issues

### Swama Connection
- Ensure Swama server is running
- Check the server URI in Settings
- Verify firewall isn't blocking localhost connections

### Model Loading
- Models are fetched from Swama automatically
- Check Swama logs if models don't appear
- Verify you have models installed in Swama

### Performance
- Ensure you're on Apple Silicon (M1/M2/M3)
- Check available system memory
- Close other resource-intensive applications
